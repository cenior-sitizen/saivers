# Xiaomi Purifier Test — Frontend

Simple test UI for the purifier API.

## Run

1. Start the backend first (see `../backend/README.md`).

2. Serve this folder:

   ```bash
   cd xiaomi-purifier/frontend
   python -m http.server 3001
   ```

3. Open http://localhost:3001 in your browser.

4. Change the API URL if your backend runs on a different host/port (default: 8002).

5. Click **Turn On** or **Turn Off** to test.
